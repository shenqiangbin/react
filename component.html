组件

将通用的代码封装成可共用的东西，避免重复编写。这就是组件。
比如，上一个的我收集的网站展示。

想要的效果
博客园
程序员的一个写博客的网站
http://www.cnblogs.com
CSDN
全球最大中文IT社区
http://www.csdn.net
每一个部分的代码如下： <div class='siteBlock'> <div class='siteBlock-img'> <img src="http://www.csdn.net/css/logo.png"> </div> <div class='siteBlock-title'> CSDN </div> <div class='siteBlock-Desc'> 全球最大中文IT社区 </div> <div class='siteBlock-url'> <a href="http://www.csdn.net">http://www.csdn.net</a> </div> </div>
***********************************************************************

实现

代码

<div id='container'></div> <script type="text/babel"> var SiteBlock = React.createClass({ render:function(){ return <div className='siteBlock'> <div className='siteBlock-img'> <img src={this.props.logo} /> </div> <div className='siteBlock-title'> {this.props.title} </div> <div className='siteBlock-Desc'> {this.props.desc} </div> <div className='siteBlock-url'> <a href={this.props.url}>{this.props.url}</a> </div> </div>; } }); var arr = new Array(); arr.push({name:'博客园',url:'http://www.cnblogs/com'}); arr.push({name:'CSDN',url:'http://www.csdn.net'}); arr.push({name:'github',url:'http://www.github.com'}); ReactDOM.render( <div> { arr.map(function(site){ return <SiteBlock logo="http://www.csdn.net/css/logo.png" title={site.name} desc="描述" url={site.url} /> }) } </div> , document.getElementById('container') ); </script>
***********************************************************************

注意

  • class属性要写成className，for属性要写成htmlFor，否则会丢失
  • 绑定属性时，不能加引号，比如：title='{site.name}'，则会当作值来显示，正确的要去掉引号：title={site.name}
  • 只能有一个最外层标签
  • 定义的组件名字，首字母必须大写，比如SiteBlock
